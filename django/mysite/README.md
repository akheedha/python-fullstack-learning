# 🌐 MySite - Django Beginner Project

## 📌 Overview

**MySite** is a beginner-friendly Django web project created to understand the fundamentals of full stack development using Django.  
It includes multiple web pages, basic routing, styling, and JavaScript interactions.

---

## 🚀 Features

### ✅ Django Setup
- Created a Django project named **mysite**
- Successfully ran the development server
- Verified setup using Django’s default page:
  👉 *"The install worked successfully!"*

## 🏠 Web Pages

### 1. Home Page (`/`)
- Displays a welcome message:  
  👉 **"Welcome to My Page"**
- Includes a styled button
- Button triggers a message using JavaScript

---

### 2. About Page (`/about`)
- Displays:
  - Heading: **"About Us"**
  - A short descriptive paragraph

---

### 3. Gallery Page (`/gallery`)
- Displays:
  - Heading: **"My Image Gallery"**
  - Three images with captions

---

### 4. Contact Page (`/contact`)
- Displays:
  - Heading: **"Contact Me"**
  - A button that shows a **Thank You message** using JavaScript
